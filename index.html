Hola, soy Merli

Soy Rita Merli Saldaña Campos, psicóloga clínica con experiencia en procesos emocionales para niñas, niños, adolescentes, adultos y parejas. Me he formado en psicología educativa, liderazgo y recursos terapéuticos basados en evidencia.

He acompañado a instituciones como UGEL 02, academias y centros de salud con talleres de desarrollo personal, emocional y liderazgo. Mi propósito es ayudarte a reconectar contigo, cultivar autoestima sólida y elegir relaciones más conscientes.

Trabajo desde Lima con modalidad online y presencial bajo solicitud. En cada encuentro utilizaremos psicoeducación, dinámicas vivenciales y rituales que devuelven calma al cuerpo.

Diseño CSS para un fondo terapéutico

Inspírate con este fondo minimalista, suave y profesional. Puedes usarlo en secciones específicas envolviéndolas en la clase therapy-background.

CSS completo

.therapy-background {
  position: relative;
  isolation: isolate;
  padding: 64px 24px;
  border-radius: 32px;
  background: linear-gradient(135deg, #f9f4ff 0%, #fef8f3 50%, #f6fbff 100%);
  overflow: hidden;
}

.therapy-background::before {
  content: "";
  position: absolute;
  inset: 12%;
  border-radius: 40px;
  background: radial-gradient(circle at top right, rgba(77, 60, 255, 0.08), transparent 55%),
              radial-gradient(circle at bottom left, rgba(255, 153, 102, 0.12), transparent 50%);
  filter: blur(0px);
  z-index: -2;
}

.therapy-background::after {
  content: "";
  position: absolute;
  inset: 0;
  background-image: url("assets/overlay-terapeutico.png");
  background-size: cover;
  background-position: center;
  opacity: 0.35;
  mix-blend-mode: soft-light;
  z-index: -1;
}

Alternativas de color

Calma Lavanda Bienestar femenino, combinación lavanda + crema.
Verde Salvia Perfecto para mensajes corporales o talleres.
Rosa cuarzo Ideal para comunicar contención y afecto.

¿Dónde colocar la imagen?

Guarda tu textura u overlay suave en assets/overlay-terapeutico.png. Puede ser un patrón en PNG (transparente) de 1200 px como mínimo para que no pierda calidad al cubrir pantallas grandes.

Si prefieres otro nombre o varias texturas, manténlas dentro de /assets y actualiza la ruta del CSS según sea necesario.
